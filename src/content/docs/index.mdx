**KiCad Library Manager (KiLM)** is a command-line tool designed to simplify the way you handle KiCad symbol, footprint, and 3D model libraries. It helps maintain consistency across different projects and workstations by providing a centralized way to manage library configurations and environment variables within KiCad.

[![PyPI version](https://img.shields.io/pypi/v/kilm.svg)](https://pypi.org/project/kilm/)
[![Python versions](https://img.shields.io/pypi/pyversions/kilm.svg)](https://pypi.org/project/kilm/)
[![License: MIT](https://img.shields.io/badge/License-MIT-blue.svg)](https://opensource.org/licenses/MIT)


## Key Features

<CardGrid stagger>
	<Card title="Cross-Platform Detection" icon="laptop">
		Automatically finds KiCad configurations on Windows, macOS, and Linux.
	</Card>
	<Card title="Centralized Management" icon="setting">
		Add symbol, footprint, and 3D model libraries to KiCad from a central setup.
	</Card>
  <Card title="Environment Variables" icon="environment">
		Set KiCad environment variables directly through the tool.
	</Card>
	<Card title="Pinned Libraries" icon="pin">
		Manage your favorite libraries easily within KiCad.
	</Card>
  <Card title="Template Management" icon="folder">
		Create and use project templates to standardize designs.
	</Card>
  <Card title="Safe Updates" icon="cloud-download">
		Creates timestamped backups before modifying KiCad configurations.
	</Card>
</CardGrid>
